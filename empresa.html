Introducción

Base De Datos

Haremos un pequeño esquema de la base de datos, de los clientes, los empleados y la empresa. En este esquema reflejaremos el material que se va a utilizar y relacionaremos a los empleados de cada departamento con el empleado al que se dirige. También hemos incluido los formularios poniendo todos los datos en las tablas. Los formularios los rellenamos con sus datos para depués insertar unos botones que sirvan: para eliminar o buscar registros, para ir al primer o al último registro, para abrir y cerrar registros o para ir al siguiente o anterior de los registros.

Arduino

Hacemos la app para poder avanzar con la programación del robot. En la app lo primero que hemos hecho es el diseño del mando que después usaremos para controlar el robot. Le ponemos los botones principales, que son de los de giro, dirección y parada, para que el robot pueda obedecer al control del mando. Hay que conectarlo con el bluetooth y una vez conseguida esta conexión, configuraremos el control del mando.

Servidor web



Primero instalamos Debian10 en la máquina virtual y pusimos dos discos para hacer el Raid1. Tuvimos que instalar Apache, MariaDB y PHP para poder configurar Nextcloud e instalarlo. Con mysql -u root -p, creamos el usuario de mysql para poder iniciar sesión en Nextcloud. Después descargamos el programa (https://download.nextcloud.com/server/ releases/nextcloud-17.0.1.zip) y lo descomprimimos (unzip nextcloud-17.0.1.zip). Por último, lo llevamos a /var/www/html/ y le damos permiso con chmod 777 en nuestro directorio.

Entramos en: nano /etc/apache2/sites-available/000-default.conf
