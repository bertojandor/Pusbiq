Introducción

Database

We will make a small scheme of the database including: clients, employees and the company. In this scheme we will state the material that is going to be used and we will create the relations between the employees of each department with the employee to whom it is directed. We have also included the forms, putting all the data in the tables. We fill in the forms with their data and then insert some buttons that will be used to manage recorded data: delete or search records, to go to the first or last record, open and close records or go to the next or previous record.

Arduino

We make the app to be able to programme the robot. The first thing we have done in the app, is the design of the controller that we will use later for the robot. We create the main buttons, which are for turning, direction and stop, so that the robot will follow the control commands. It is a bluetooth connection and once this connection is achieved, we will configure the remote control.
