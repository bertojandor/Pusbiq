Introducción


En esta página web explicaremos el procedimiento para superar el reto del robot. Lo primero es crear una red inalámbrica para poder proteger nuestro sistema. Después diseñaremos algunas figuras para nuestra ciudad robótica y programaremos un robot para luchar contra otros equipos. Nuestro equipo se llama Pusbiq.


Para decorar la ciudad, elegimos hacer una barrera automática y unas casas. Para poder conectarnos al móvil de manera segura, crearemos una subred del wifi con contraseña y así evitaremos que nos hackeen. Desarrollaremos la web en HTML, utilizaremos freecad para el diseño 3D y usaremos arduino para programar el robot que luchará contra los otros equipos. Controlaremos el robot con una app en el móvil.


En cuanto a la organización grupal, comenzamos repartiéndonos el trabajo, pero nos ayudábamos entre nosotros si había alguna duda. También nos intercambiábamos el trabajo para no hacer todo el rato lo mismo y tener otras perspectivas.

¿En qué consiste?

Nuestro proyecto consiste en crear una ciudad robotizada en la que tendrá lugar una pelea de robots. La ciudad se construirá entre todos los grupos participantes y los robots creados para la pelea tendrán también que interactuar con esa ciudad.
